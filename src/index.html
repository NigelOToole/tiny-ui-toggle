Installation

$ npm install tiny-ui-toggle --save-dev

Usage

Import JS

The script is an ES6(ES2015) module but the compiled version is included in the build as "src/scripts/tiny-ui-toggle-umd.js". You can also copy "src/scripts/tiny-ui-toggle.js" into your own site if your build process can accommodate ES6 modules.

import Toggle from 'tiny-ui-toggle.js';

// Init with string - default setup for one instance
const toggleDefault = Toggle({ selector: '.toggle' });


// Init by loop - this wont allow access to methods
const toggleElements = document.querySelectorAll('.toggle');   

for (const item of toggleElements) {
  Toggle({ selector: item });
}


// Init with all options at default setting
const toggleDefault = Toggle({
  selector = '.toggle',
  activeClass = 'is-active',
  animClass = 'is-anim',
  toggleHeight = true,
  textActive = '',
  textInactive = '',
  closeAuto = false,
  closeDelay = 500
});

Options

Property Default Type Description
selector '.toggle' String Element selector.
activeClass 'is-active' String CSS class when element is active.
animClass 'is-anim' String CSS class to transition the video overlay between states.
toggleHeight true Boolean Animate the panel height.
textActive '' String Text in element when it is active.
textInactive '' String Text in element when it is inactive.
closeAuto false Boolean Automatically close the panel after a timeout.
closeDelay 500 Integer(ms) Amount of time a user leaves an element for it to close.

API

Property Type Description
instance.toggle() Method Toggles the state of the element and any target elements.
instance.toggleState() Method Toggles the state of the element.
instance.setState() Method Sets the element state.
instance['elements'] Object Returns the elements used by this instance.

Import Sass

@import "node_modules/tiny-ui-toggle/src/styles/tiny-ui-toggle.scss";

Markup

<button class="toggle toggle-btn" data-toggle-target=".demo-single" aria-expanded="false">Single panel</button>

<div class="toggle-panel demo-single" aria-hidden="true">
  <div class="toggle-panel-content">
    ...
  </div>
</div>

Events

A 'toggle' event is started once an element is toggled. The event is ended once the element has changed state.

document.querySelector('.toggle-btn-default').addEventListener('toggle', (event) => { 
  console.log(`Action: ${event.detail.action}`);
});

Compatibility

Supports all modern browsers (Firefox, Chrome, Safari and Edge) released as of October 2021.
